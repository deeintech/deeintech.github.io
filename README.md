# Dee's blog

## Install jekyll, ruby

```
https://jekyllrb.com/docs/installation
```

## Install dependencies

```
bundle
```

# Run locally

```
jekyll serve
```

# Deploy to prod

make updates in the gemfile

1. comment: `# gem "jekyll", "~> 4.4.1"`

2. uncomment: `gem "github-pages", group: :jekyll_plugins`
